# Музыка
[в главный README](../../../readme.md)

Музыка - это сервис для выкладывания и прослушивания чужих песен. Этот сервис позволяет делится своими песнями, слушать и оценивать чужие.

Возможности:
- поиск музыки локально по базе
- прослушивание своих, чужих и песен друзей, загруженных на сервис
- скачивание любой музыки к себе
- создание в редакторе бита и публикация музыки
- создание и редактирование плейлистов
  - оценивание чужих песен

Обсуждения своих треков, битов, их оценки и просто общения есть [сообщество](https://t.me/saniok_develop_community)
