# Аккаунты
[в главный README](../../../readme.md)

Аккаунт - это сервис, который является сборником информации о пользователе

Возможности:
- Можно редактировать разные поля аккаунта
- Разную информацию о аккаунте можно скрывать ото всех, ото всех кроме друзей или всем показывать
- 
- Картинка аккаунта содержит изначальное стандартную картинку, но её можно поменять. Также есть оформление картинки, стили этого оформления можно менять

Текст аккаунта:
- Название аккаунта (изначально устанавливается имя аккаунта из телеграмма)
- Описание аккаунта (изначально пустой блок)
- Номер телефона (изначально пустой блок)

> ✨PRO | С подпиской есть больше вариантов оформления
